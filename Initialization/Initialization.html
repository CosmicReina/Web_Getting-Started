Emmet Snippet Learning

HTML Snippet

.class : Tạo div class.

#id : Tạo div id.

.class1.class2 : Tạo div từ hai class trở lên.

.class1+.class2 : Tạo hai div cùng cấp cóclass khác nhau.

.classParents>.classChild : Tạo div khác cấp nhau.

.classTop>.classTopChild^.classBottom>.classBottomChild : Tạo div cùng cấp nhau có div dưới cấp.

div*5 : Tạo nhanh 5 div cùng cấp giống nhau.

.class$*10 : Tạo nhanh 10 div cùng cấp có tên class từ class1 đến class10.

.class{content} : Tạo div có tên class kèm nội dung bên trong
